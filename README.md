# 🏗️ Corporate Construction Portfolio Website (my-website)

> A modern, fully responsive frontend showcase website designed for a corporate construction and engineering firm to highlight core services, active/completed structural projects, key industry partners, and essential corporate capabilities.

🌐 **[View Live Demo](https://dufaxd-001.github.io/my-website/)**

---

## ✨ Features

* **Project Portfolio Showcase:** An organized, visual gallery displaying completed and ongoing structural projects with specific engineering details.
* **Corporate Overview & Services:** Clear sections detailing company capabilities (e.g., project management, civil construction, site supervision, and design).
* **Strategic Partners & Clients:** A dedicated layout highlighting corporate alliances, sub-contractors, and trusted industry partners.
* **Interactive Contact & Inquiry:** A clean, professional contact section with embedded location mapping and data fields for corporate inquiries.
* **Responsive Layout:** Optimized specifically for corporate clients reviewing portfolios on tablets, mobile devices, or desktop screens.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6)
* **UI/UX Template Design:** canva
